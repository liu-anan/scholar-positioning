## Template rules

- Keep URLs or page names only where they help later auditing
- Keep `Representative Works` phase-aware
- Keep `Genealogy / Context Clues` separate from `Method Clues`
- Keep `Direct neighbors` separate from `Adjacent routes`
- Fill `Claims to avoid` before drafting the final profile

## Minimum usable pack

A source-pack is minimally usable when it contains:

- stable identity
- a cross-phase representative work set
- at least rough phase clues
- at least one method-combination clue
- at least provisional neighbor structure
- explicit claims to avoid

If one of these is missing, run the checklist before drafting and decide whether to search more or downgrade the profile.
