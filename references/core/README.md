# Core Layer

This folder is the stable protocol layer.

Keep here:

- artifact model
- execution rules
- evidence / coverage protocol
- source-pack workflow
- retrieval workflow
- roadmap derivation rules
- eval checklists

These files should change slowly and should not depend on one user or one domain.

Current note:

- the public package is still in a compatibility-first layered stage
- top-level `references/*.md` remain the current live protocol entry files
- this folder describes the intended stable home for protocol assets
