# Personalize 示例

这个目录放的是最小 personalize 示例。

用途：

- 给第一次使用这个 skill 的人一个可直接参考的起点
- 帮助团队成员快速理解“什么适合放在 user overlay，什么不适合”
- 降低第一次配置成本

这些示例的定位是：

- 只改呈现偏好
- 不改核心协议
- 不改 artifact model
- 不改 evidence / coverage / router

推荐使用方式：

1. 找一个最接近自己的示例
2. 复制成自己的 `preferences.md`
3. 只改语言、摘要、阅读顺序、默认任务倾向、主领域等内容
4. 不要把核心协议改进 user overlay
